Methodologies for Text Claude Analysis

Architecture is theoretical; performance is empirical. How do we actually measure "thinking"?

Quantitative Metrics

  • Reasoning Depth Score (RDS): Custom metric (0‑10) capturing the number of successful internal passes.
  • Latency (ms): Time from prompt submission to final extended thinking output.
